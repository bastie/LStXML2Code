# ``LStXML2Code``

LStXML2Code bringt die XML Programmablaufpläne in eine allgemeine abstrakte Baumstruktur.

## Overview

LStXML2Code ist ein Framework, welches ermöglicht die XML-Programmablaufpläne zu verarbeiten.
